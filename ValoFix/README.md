## VALOFIX

VALORANT 定点、共通マクロ管理プログラム

### VALOFIX とは？

FPS ゲーム「VALORANT」のエージェント（キャラクター）ごとの定点や共通マクロを説明と URL を合わせて登録し、いつでも情報を素早く見返せるようにするのを目的に開発されたプログラムです。

### 用語説明

**定点**：VALORANT には「この位置に立って、この角度でアビリティ（爆弾など）を投げたら必ず同じ場所に着弾する（風の強さ、エージェントの体調などの概念が存在しないため）」という法則があります。これらの法則を利用してゲームを優位に進められるスキルの投げ方をまとめて「定点」と呼びます。
**マクロ**：それぞれのマップ（ステージ）における知識のことです。「ここに立っていれば敵にバレにくい」「ここに立っていると敵から足が見えてしまう」など

### 主な機能

・ユーザー登録、ログイン
・マップ、エージェント選択
・定点、マクロの表示と新規登録

### プログラム操作確認をする際は･･･1.セットアップ方法

1.XAMPP の htdocs/に配置 2.「valofix」という名前の MySQL データベースを作成
3.valofix.sql をインポート 4.ブラウザで「http://localhost/valofix/index.php」にアクセス
※valofix.sql の「agents」テーブルにおいて agent_id=25 のレコードが存在しないのは仕様であり、データの破損などではございません。

### プログラム操作確認をする際は･･･2.サンプルユーザーについて

### プログラム操作確認をする際は･･･3.操作していただきたいこと

1.サンプルユーザーでログイン 2.左側のサイドバーにあるフォームでマップ →「ABYSS」、エージェント「BRIMSTONE」を選択して右のマップ画像に表示される 4 つの白い円にそれぞれマウスホバーしてください。そのページ（main.php）における定点、マクロ情報が登録されております。「共通マクロ」については、サイドバーのフォームでマップを再び「ABYSS」、エージェントは「BRIMSTONE」以外の任意のエージェントを選択して送信を押していただければ、「BRIMSTONE」限定の「定点」の情報は非表示になり、共通マクロの情報だけが表示されることがご確認できます。 3.マップ画像の任意の位置をクリックすると新規定点、マクロ登録フォームのポップアップがクリック位置付近に表示されます。フォームの定点、共通マクロを選択するラジオボタンがありますが、どちらかを選んでコメントと URL に適当な文字列を入力した後、登録ボタンを押して入力した情報が新規に登録されていることをご確認ください。 4.サイドバーのフォームでマップ →「LOTUS」、エージェント →「VYSE」には、実際のゲームに存在する定点やマクロの情報を登録しております。

### 画面遷移フロー

1.index.php(トップページ)：ユーザー登録済みなら 2.に遷移、そうでないなら 3.に遷移。
2.login.php(ログインページ)：ユーザーログインを行う。ログイン後、4.に遷移。
3.regist.php（ユーザー新規登録ページ）：ユーザー新規登録を行う。登録後、2.に遷移してログインを行う。
4.main.php（定点、マクロ表示・登録ページ）：自分で登録した定点、共通マクロを表示する。マップ画像上の任意の位置をタップして新規登録が可能。
